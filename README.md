# Customer Revenue & Retention Analysis

## Overview

This project analyzes customer transaction data to understand revenue distribution and retention patterns.

## Objectives

* Analyze revenue contribution across customers
* Evaluate customer retention (repeat vs one-time)
* Segment customers based on value
* Understand purchasing behavior

## Key Insights

* A small group of customers drives most revenue
* Many customers only make a single purchase
* Frequent customers generate higher value

## Business Impact

* Retaining high-value customers is critical
* Increasing repeat purchases boosts revenue
* Customer behavior affects long-term growth

## Tools & Techniques

* SQL (Google BigQuery)
* Aggregation
* CTE (Common Table Expressions)
* Window Functions

## Project Structure

* `queries/` → SQL analysis
* `presentation/` → Slide deck
